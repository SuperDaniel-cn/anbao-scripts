# 多平台视频上传助手

一个强大的多平台视频上传自动化脚本，支持一键将视频发布到多个主流社交媒体平台。

## 🚀 支持的平台

- **B 站 (bilibili)** - 支持分区选择、定时发布
- **抖音 (douyin)** - 支持话题标签、发布选项
- **快手 (kuaishou)** - 支持视频上传和发布
- **小红书 (xiaohongshu)** - 支持图文视频发布
- **视频号 (weixin)** - 微信视频号发布

## ✨ 主要功能

- 🎬 **批量上传** - 一次操作，多平台发布
- 📋 **智能表单** - 根据平台动态显示相关选项
- ⏰ **定时发布** - 支持设置发布时间
- 🖼️ **封面上传** - 支持自定义视频封面
- 🏷️ **标签管理** - 支持话题标签和分类
- 📝 **草稿保存** - 可选择存为草稿或人工审查
- 🔄 **自动重试** - 上传失败自动重试机制

## 📋 使用说明

### 基本参数

1. **视频文件** - 选择要上传的视频文件
2. **封面文件** - 选择视频封面图片
3. **视频标题** - 填写视频标题
4. **视频简介** - 填写视频描述内容
5. **话题标签** - 添加相关话题标签（用逗号分隔）
6. **定时发布** - 设置发布时间（可选）
7. **最终操作** - 选择发布、存草稿或人工审查

### 平台专属参数

- **B 站分区** - 选择视频所属分区（仅 B 站显示）
- **上传超时** - 设置上传超时时间（高级选项）

## 🛠️ 安装与使用

### 安装步骤

1. 在安宝助手平台中导入脚本
2. 提前在安宝助手里配置各平台的登录凭证
3. 填写表单参数并运行脚本

### 使用流程

1. 选择目标平台（根据凭证自动匹配）
2. 选择要上传的视频文件和封面
3. 填写视频标题、简介和标签
4. 设置发布选项（立即发布/定时发布/草稿）
5. 点击运行，脚本将自动完成上传流程

## 🔧 高级配置

### 浏览器选项

脚本使用 Chrome 浏览器运行，支持以下配置：

- 有头模式：便于观察和调试
- 慢速执行：模拟人工操作，降低检测风险

### 超时设置

- 默认上传超时：30 分钟
- 可根据网络情况调整超时时间

## 📝 注意事项

1. **文件格式** - 支持常见视频格式（MP4、AVI、MOV 等）
2. **文件大小** - 建议单个视频文件不超过平台限制
3. **网络环境** - 确保网络稳定，大文件上传需要较长时间
4. **平台规则** - 请遵守各平台的内容规范和上传限制
5. **账号状态** - 确保各平台账号已正常登录且无异常

## 🐛 故障排除

### 常见问题

**Q: 上传失败怎么办？**  
A: 检查网络连接、文件格式和账号状态，可适当增加超时时间。

**Q: 某个平台上传失败？**  
A: 可能是平台临时维护或接口变更，建议稍后重试或查看平台公告。

**Q: 脚本运行很慢？**  
A: 大文件上传需要时间，请耐心等待。可在设置中调整慢速执行参数。
