鉴权网址
https://me.weibo.com/
验证是否有这个元素getByText('昨日阅读')
验证成功之后，goto https://weibo.com/upload/channel

上传视频的input 是这个
<input type="file" accept="image/*, .jpg, .jpeg, .bmp, .gif, .png, .heif, .heic, video/mp4,video/x-m4v,video/*,.mkv,.flv" multiple="multiple" class="FileUpload_file_27ilM">

会有getByText('上传中').first()的状态

上传完成之后getByText('上传完成').first()会显示上传完成

选择类型：
  await page7.getByText('原创').click();
  await page7.getByText('二创').click();
  await page7.getByText('转载').click();
这三个类型是互斥的，只能选一个

getByRole('textbox', { name: '填写标题（0～30个字）' })
标题输入框，点击后可以输入标题

下面开始上传封面：
点击这个裁剪封面page8.getByRole('link', { name: '裁剪封面' }).click();
验证是否有getByText('编辑封面')
验证成功之后，<input type="file" accept=".jpg, .jpeg, .bmp, .gif, .png" class="VideoEdit_file_2aoWb">用这个元素上传封面
然后getByRole('button', { name: '完成' }).click();
点击完成

设置微博描述：
getByRole('textbox', { name: '有什么新鲜事想分享给大家？' }).click()
点击这个可以输入描述


输入完描述之后直接输入标签
微博的话题标签是【#职场话# 】两个#中间放标签内容，然后后面的#后面要有一个空格来间隔其他标签

输入完之后getByRole('button', { name: '发布' }) 即可发布
